This website has been designed to share information about Data Science. There are two main topics: machine learning and artificial intelligence, both explained along with real world examples.

In order to unleash the power of R, the website is entirely built in R with blogdown. R is the best data specific programming language focused around data analyst and data scientist kind of process. It is also one of the faster ways to build dashboards and web apps.

The take home message is: useR! and do not forget that all models are wrong, some models are useful.
